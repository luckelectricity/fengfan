# 注册
## 用户添加
#### 请求地址
    /api/users/add
#### 请求方式
    POST
#### 请求参数
    {
      username: 'zhangsan', // [字符串：必填]
      password: 'abc123' // [字符串：必填]
    }
#### 返回数据
	{
      "errcode": 0, // 错误代码：[数值：必填] 0 无错误 -1 有错误
      "errmsg": "", // 错误信息：[字符串：默认为空]
      "data": { // 数据内容
        status：'ok', // 存取状态：[字符串：必填] 'ok' 成功 'fail' 失败
        msg: '数据存取成功' // 附加信息：[字符串：选填]
      }
	}
